# Aegis Bank Web Client

This is the modern web portal client for the Aegis Secure Banking Platform. Built with Next.js, React, TailwindCSS, and SWR.

## Key Features
- High-fidelity visual user interface with Inter and Outfit fonts, customized styling and smooth transitions
- Multi-currency debit and savings account balances
- Security Control Room allowing developers to turn on/off defensive shields (SQLi, IDOR, Parameter Tampering)
- Persistent user sessions on page reload (F5)
